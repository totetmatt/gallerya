# Gallerya
Image resizer and static gallery generator

# Installation

# Usage 
```
  -author string
        Author name
  -base-url string
        Full url of the gallery (default "/")
  -description string
        Small description
  -facebook string
        Facebook name
  -github string
        Github name
  -instagram string
        Instagram name
  -linkedin string
        Linkedin name
  -original string
        Original Photo directory (default "./original")
  -skip-image-thumb-processing
        Skip the image transformation
  -static string
        Path to Static File (default "./static")
  -thumbnail string
        Thumbnail directory (default "./thumb")
  -title string
        Title of the gallery (default "Gallerya")
  -twitter string
        Twitter name
  -workers int
        Number of workers (default 4)
```
